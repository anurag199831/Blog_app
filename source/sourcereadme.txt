Basics Of How To Host :-
1. Get Source code from the above mentioned git link
2. Install Software mentioned in the requirement.txt present inside source directory
3. Go to source directory
4. Run command "python3 manange.py runserver"
5. Now browse to the URL mentioned in the output of above command.

For Login :-
Click the login button present at homepage

For Selecting your interested category click on Edit Category:-
Click on Edit Category and select your interested categories.

For Adding post :-
Click on the Add Post and Select the category related to post.
Then type your blog and add image or videos(if any).
